# 🤖 AI Code Assembler Bot

Telegram бот для сборки кода из частей с помощью AI.

## Команды
- `/start` — начать
- `/show` — показать код
- `/done` — скачать файл
- `/reset` — очистить
- `/new_project` — создать проект
- `/files` — список файлов
- `/complexity` — анализ сложности
- `/bugs` — поиск багов
- `/validate` — проверка
- `/web` — веб-редактор
